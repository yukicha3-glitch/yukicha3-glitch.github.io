画面録画ツール

OBS風の画面録画アプリ。シーン合成・ライブプレビュー・ホットキー・リプレイバッファに対応。完全ローカル・無料ライブラリのみ。

Windows 10/11 シーン合成 マルチトラック音声 登録不要
⬇ ダウンロード（ZIP / 約118MB）

ZIPを展開して ScreenRecorder.exe を実行。インストール不要。

特徴

  • シーン合成（OBS風） — 画面 / ウィンドウ / カメラ / 画像 / テキストを1枚のキャンバスに重ねて録画。ウィンドウは裏に隠れても録画継続。
  • ライブプレビュー — ソースをドラッグで移動、四隅でリサイズ。端・中央への吸着（スナップ）とガイド線できれいに整列。
  • マルチトラック音声 — システム音声・マイク・アプリ別の音を自由に組み合わせ。Tr1〜Tr6にトラック分け可能（48kHz ステレオ AAC）。
  • アプリ別音声キャプチャ — 選んだアプリ（と子プロセス）の音だけを録音（WASAPIプロセスループバック）。
  • 音声のみ録音 — 映像なしで m4a / mp3 / wav に収録。ラジオ・会議録音にも。
  • リプレイバッファ — 直前のN秒（5〜600秒）を常時バッファし、ワンキーで書き出し。神プレイの後追い保存に。
  • ホットキー — Ctrl+F9 録画開始/停止、Ctrl+F10 一時停止/再開、Ctrl+F11 リプレイ保存（裏でも有効・管理者権限不要）。
  • 保存先ブックマーク — よく使うフォルダを★登録してワンクリック切替。

動作環境

OSWindows 10（build 20348+）/ Windows 11 64bit
インストール不要（ZIP展開してexe実行）
出力形式MP4 / MKV、音声 AAC（m4a/mp3/wav）
ネット通信なし（完全ローカル）

使い方

  1. ZIPを展開し、ScreenRecorder.exe をダブルクリック。
  2. 「+ ソースを追加」で録りたい画面・ウィンドウ・カメラを追加。
  3. 「+ 音源を追加」でシステム音声・マイク・アプリ音を追加。
  4. 保存先を選び、Ctrl+F9 または録画ボタンで開始/停止。
アプリ別音声キャプチャは Windows 10 build 20348 以降 / Windows 11 が必要です。
