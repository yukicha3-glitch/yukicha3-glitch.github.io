APIキーマネージャー

各種サービスのAPIキーをローカルで安全に管理するツール。Argon2id で鍵導出し AES-256-GCM で暗号化。通信なし・EXE単体でポータブル。

Argon2id AES-256-GCM 通信なし ポータブル
⬇ ダウンロード（EXE / 約40MB）

APIManager.exe をダブルクリックで起動。インストール不要。

セキュリティ設計

鍵導出Argon2id
暗号化AES-256-GCM（認証付き暗号）
Vault形式単一暗号化ファイル、原子的書き込み。salt/nonce は都度再生成
マスターPW保存（任意）Windows DPAPI
クリップボードコピー後に自動消去
通信一切なし

特徴

  • APIキーを一元管理 — サービス名・キー・メモをまとめて整理。
  • 強力な暗号化 — パスワードマネージャーと同じ Argon2id + AES-256-GCM 基盤。
  • 完全ポータブル — EXE単体、データもexeの隣。USBで持ち運び可能。
  • ネットワーク通信なし — APIキーが外部に出ることはありません。
  • フォルダ分け — ツリー構造で整理。

動作環境

OSWindows 10 / 11 64bit
インストール不要（約40MBの単一exe）
ネット通信なし

使い方

  1. APIManager.exe を起動。
  2. 初回はマスターパスワードを設定。
  3. サービス名とAPIキーを登録。必要なときにコピーして使用。
⚠ OS そのものが侵害された環境では安全を保証できません。マスターパスワードを忘れるとデータは復号できません。
