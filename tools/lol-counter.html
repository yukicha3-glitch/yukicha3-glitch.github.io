LoL カウンターピック分析

レーンとチャンピオンを選ぶと、OP.GG／DEEPLOL の統計からカウンターピック・有利マッチアップ・勝率を表示。TFTメタ構成タブも搭載。すべて無料・登録不要・Riot APIキー不要。

OP.GG / DEEPLOL APIキー不要 TFT構成 ポータブル
⬇ ダウンロード（ZIP / 約41MB）

ZIPを展開して LoLCounterPick.exe を実行。Python不要・インストール不要。

特徴

  • カウンター分析 — チャンピオンを選ぶと、苦手な相手（カウンター）と有利な相手を勝率付きで表示。
  • レーン自動選択 — そのチャンピオンの最多使用レーンが自動で選ばれる（例: ヤスオ→ミッド）。
  • ローマ字入力対応 — IMEを使わず gare→ガレン のように入力可能。Tab で候補移動、Enter で確定。
  • 情報源・リージョン切替 — OP.GG（Global/JP/KR）／ DEEPLOL（KR）。★=OP.GG認定カウンター。
  • 詳細表示 — 勝率・ピック率・バン率・ティア・おすすめスペル／コアアイテム（日本語名）。
  • TFT構成タブ — MetaTFT（統計約70件）と TFTips（日本語ガイド・盤面画像付き）。ゲーム中向けの常時最前面ミニ表示も。
  • ジャンプ＆戻る — 表のチャンピオンをクリックで分析にジャンプ、「← 戻る」で直前に戻る。

動作環境

OSWindows 10 / 11 64bit
インストール不要（ZIP展開・ポータブル）
ネット接続必要（統計データの取得に使用）
データ保存初回にデータ/アイコンを cache/ へDL。2回目以降は高速
データソースOP.GG・DEEPLOL・MetaTFT・TFTips の内部API、Riot Data Dragon（すべて無料・登録不要）

使い方

  1. ZIPを展開し、LoLCounterPick.exe をダブルクリック（初回はデータDLで少し時間がかかります）。
  2. チャンピオン名を入力して候補から選ぶ（選ぶと即分析）。
  3. レーン・ティア・情報源を切り替えると自動で再分析。
初回起動時に Windows SmartScreen の警告が出ることがあります（署名のない自作アプリのため）。「詳細情報」→「実行」で起動できます。アンインストールは exe と cache フォルダを削除するだけです。
