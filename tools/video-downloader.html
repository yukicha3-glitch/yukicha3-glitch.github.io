動画ダウンローダー

URLを貼るだけで動画をダウンロード。エンジンは yt-dlp、GUIは PyQt5。YouTube・Twitch など yt-dlp 対応の1000以上のサイトに対応。

Windows 10/11 yt-dlp 画質選択 複数同時DL
⬇ ダウンロード（EXE / 約54MB）

VideoDownloader.exe をダブルクリックで起動。Python不要・ポータブル。

特徴

  • URLを貼るだけ — 1行に1つ、複数まとめてOK。プレイリストURLにも対応。
  • 保存形式が豊富 — 動画は MP4 / MKV / WEBM、音声は MP3 / M4A / WAV / FLAC / Opus。音声形式を選ぶと自動で音声だけ抽出。
  • 画質選択 — 最高画質 / 1080p / 720p / 480p。
  • 複数同時DL — 1〜6本を並列ダウンロード。
  • 保存先ブックマーク（名前付き） — 「アニメ」「音楽」などフォルダに名前を付けてワンクリック切替。
  • 進捗表示 — 速度・残り時間・状態を行ごとに色分け表示。
  • クリーンなキャンセル — 停止すると未完成の .part 等を自動削除。フォルダにゴミが残らない。
  • 履歴タブ — DL履歴から再生・フォルダを開く・再ダウンロードが可能。

動作環境

OSWindows 10 / 11 64bit
インストール不要（exe単体・ポータブル）
必要なものffmpeg（高画質の映像+音声結合・MP3変換に使用。インストール済みなら自動検出）
データ保存settings.json / history.json をexeの隣に作成（フォルダごと移動可）

使い方

  1. VideoDownloader.exe を起動。
  2. URLを貼り、画質・保存先・同時DL数を選ぶ。
  3. 「＋ キューに追加」→「▶ ダウンロード開始」。
DLに失敗するようになったら、yt-dlp を最新版に更新してください（サイト側の仕様変更に追従するため）。各サイトの利用規約・著作権の範囲内、私的利用の範囲でご利用ください。
