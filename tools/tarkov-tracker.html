Tarkov タスクトラッカー

Escape from Tarkov の全タスクを管理。完了チェック、必要アイテムの自動集計、全12マップへのタスク目標マーカー表示。日本語／English切替対応。すべて無料・登録不要・APIキー不要。

tarkov.dev API 全12マップ対応 日本語/English ポータブル
⬇ ダウンロード（ZIP / 約40MB）

ZIPを展開して TarkovTracker.exe を実行。Python不要・インストール不要。

特徴

  • 全タスク一覧 — 全510以上のタスクをトレーダー別に表示。ダブルクリックで完了チェック、検索・マップ・カッパ・レベルでフィルタ。
  • カッパ対応 — Kappa（カッパコンテナ）に必要なタスクを★表示、達成率をステータスバーに常時表示。
  • 必要アイテム集計 — 未完了タスクで必要な納品アイテムを自動集計。FIR（レイド中入手）が必要な個数も一目で分かる。
  • マップ表示 — Customs、Woods、Shoreline から The Lab、Terminal まで全12マップにタスク目標の正確な位置を色分けマーカーで表示。ズーム・パン・ホバーで詳細。
  • タスク詳細 — 目標一覧・必要アイテム・前提タスク・獲得XP。公式Wikiへワンクリックでジャンプ。
  • 日英切替 — タスク名・目標・アイテム名・UIすべて日本語／英語を切替可能。
  • ゲームアップデート対応 — 「データ更新」ボタンで最新のタスクデータを再取得。

動作環境

OSWindows 10 / 11 64bit
インストール不要（ZIP展開・ポータブル）
ネット接続初回とデータ更新時のみ必要（約20MBダウンロード）。以降はオフライン動作可
データ保存進行状況は exe と同じフォルダの progress.json に保存
データソースtarkov.dev（the-hideout プロジェクト、無料・登録不要）

使い方

  1. ZIPを展開し、TarkovTracker.exe をダブルクリック（初回はデータDLで1〜2分かかります）。
  2. タスクタブで完了したタスクにチェック。「未完了のみ」で今やるタスクだけ表示。
  3. 必要アイテムタブで集めるアイテムを確認、マップタブで目標の位置を確認してレイドへ。
初回起動時に Windows SmartScreen の警告が出ることがあります（署名のない自作アプリのため）。「詳細情報」→「実行」で起動できます。アンインストールは exe と cache フォルダを削除するだけです。本ツールは Battlestate Games 非公式のファンツールです。
